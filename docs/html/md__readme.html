About the program

This program compares the performance of Dijkstra‚Äôs algorithm and A* for computing shortest paths on random geometric graphs.

  • Generate n random points in the square ([0, 100]^2)
  • Build a k‚Äìnearest neighbor graph using Euclidean distances between points
  • Choose the source s and target t based on lexicographic order
  • Run Dijkstra‚Äôs algorithm from s to t
  • Run A* from s to t using the Euclidean distance to t as the heuristic (h(u))

  • Analyze and compare the results
